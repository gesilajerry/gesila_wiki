# 概念卡：霍尔木兹从"概率风险"到"已实现损失"

> 归档时间：2026-04-26 | 分类：地缘政治 × 能源安全 | 来源：推送日报+会话复盘

---

## 背景

霍尔木兹海峡是全球约21%的石油运输咽喉要塞。2026年4月25-26日，伊朗最高领袖下令"不得恢复战前状态"，特朗普取消谈判行程，油价触及$107（年内新高）。这不是"可能发生的风险"，而是**正在产生实际经济损失的已实现损失**。

这一转变具有重大分析意义：此前市场将霍尔木兹风险视为概率折价（油价含地缘风险溢价约5-8%），而当前溢价已实质性地计入供需缺口。

---

## 核心洞察

### 1. 三维定价机制已重构

**地缘溢价 → 供需缺口溢价：**
- 此前：风险概率 × 预期损失 = 地缘溢价（理论值）
- 当前：实际运输量下降 + 库存消耗 = 供需缺口已实现
- 结果：IEA最高级别警告 → 溢价从"可能性"变成"现实成本"

### 2. 传导链：从能源到半导体制造

```
油价$107 → 电力成本↑ → 芯片制造成本↑ → IDM/代工厂毛利承压
         → 炼化成本↑ → 化工品价格↑ → 中游材料成本↑
         → 运输燃料↑ → 物流成本↑ → 终端消费品价格压力
```

### 3. A股联动逻辑确认

- 能源板块（中国石油、中国神华、陕西煤业）：直接受益
- 化工板块（恒力石化等）：成本上升 + 制裁双重打击
- 半导体板块：电力成本传导，成熟制程代工厂（UMC等）成本压力显现

---

## 对Jerry的意义

1. **投资组合层面**：持仓中的能源股（若持有）迎来实质性利好；但需警惕化工链的制裁延伸风险（恒力模式可能复制）
2. **产业研究层面**：霍尔木兹不再是"地缘风险提示"，而是**已发生的供应链压力测试**，其持续时间将成为A股能源板块和半导体板块盈利预测的关键变量
3. **情报追踪层面**：4月27日美伊谈判重启是下一个核心观测节点，结果将决定$107是底部还是新一轮上涨的起点

---

## 延伸思考

**关键判断框架转变：**
- 旧框架：霍尔木兹风险 = 地缘政治风险 → 分析"会不会发生"
- 新框架：霍尔木兹损失 = 已实现供应链冲击 → 分析"持续多久"和"如何传导"

这一转变同样适用于其他地缘事件（如台海、朝鲜半岛），当事件从"概率"变成"现实"，传统的风险折价模型需要更新为**损失半径估算模型**。
