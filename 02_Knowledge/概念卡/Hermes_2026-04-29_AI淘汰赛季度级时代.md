# AI淘汰赛进入"季度级"时代

**背景**
2026年4月，GPT-6 预训练完成（预计6月开放 API），这一节点将 GPT-5 到 GPT-5.5 的3个月间隔、GPT-5.5 到 GPT-6 的2个月间隔压缩为"季度迭代"——比2024年的"年度迭代"快了整整4倍。Hermes 在4月28日的结构层提炼中首次系统命名了这一现象。

**核心洞察**

从 GPT-5 到 GPT-6 的迭代轨迹揭示了 AI 竞争的核心规律：算力供给 → 模型能力 → 应用层迭代的周期正在系统性缩短。GPT-4（2023年3月）→ GPT-4o（2024年5月，14个月）→ GPT-5（2025年8月，15个月）→ GPT-5.5（2025年11月，3个月）→ GPT-6（2026年4月，2个月）。每隔一个版本，迭代速度翻倍。

这一加速有三重驱动力：(1) 算力成本曲线持续下陡，训练一个新模型的边际成本相对下降；(2) 基建-应用反馈循环压缩了"模型发布→生态验证→下一版本优化"的路径；(3) 地缘竞争（美国 vs 中国 vs 中东）使头部玩家宁可迭代快、不可迭代慢。

对投资和市场判断而言，这意味着"AI 赛道押注"的窗口期从年度压缩到季度：上一个季度还领先的供应商，下一个季度可能已被颠覆。中际旭创（A股第二只万亿级算力硬件股）的 262% 增速能否持续，需要用"季度级"视角重新评估——不是问"今年能不能维持"，而是问"这个季度会不会被新一代 HPC 架构颠覆"。

**对Jerry的意义**

他运营的财经/AI 日报需要反映这个加速节奏。当 GPT-6 预计6月开放 API 时，5月就是市场"预期定价"窗口——相关概念股、算力硬件、推理基础设施都会提前反应。日报中的"今日关注"和"核心事件"标签应增加"季度级催化"维度，帮助Jerry在纷杂的信息流中识别哪些是真正影响本周/月格局的信号。

**延伸思考**

"季度级"迭代对个人技能学习也提出了新要求：不再是"学会 XX 框架可以用一年"，而是"每季度重新评估 XX 框架是否还值得投入"。这要求建立自己的"AI 技能半衰期"判断框架——哪些技能（Python、提示工程、产品思维）是跨周期的常青能力，哪些（特定 API 调用、特定模型微调）是会在12-18个月内被工具替代的短期技能。

**标签**：AI × 竞争格局 × 投资框架 × 迭代速度 × GPT-6
