# AI军备竞赛进入爆发期：10天10个大模型

**背景**
2026年5月2日的AI热点推送汇总了多个关键信号：10天内全球连发10个大模型（80%来自中国）、GPT-6发布、DeepSeek V4发布、英伟达Nemotron全模态模型发布、五角大楼与AI巨头签署国防合作。这些信号汇聚成一个清晰的产业判断。

**核心洞察**

**核心信号：10天10个大模型，中国占80%**

Kimi K2.6、Qwen3.6、蚂蚁Ling 2.6 Flash、DeepSeek V4密集发布。这是AI发展史上从未有过的大模型发布密度，标志着基础模型能力竞争进入白热化阶段。

**两条主线**

主线1：算力瓶颈与需求爆发并存
- 英伟达全模态模型Nemotron发布，定位企业级AI智能体感官大脑，效率提升9倍
- 四巨头（谷歌、微软、亚马逊、Meta）同日宣布自研AI芯片战略，英伟达唯一供应商格局生变
- 台积电3nm/2nm产能被疯抢，HBM4竞赛白热化
- 台积电宣布2026年在美国量产3nm，总投资400亿美元

主线2：国产替代从可用向好用跨越
- DeepSeek V4 + 华为昇腾全栈生态完成布局
- 国产AI全栈生态：从芯片（昇腾）到模型（DeepSeek）到应用（600+智能应用落地）
- 斯坦福HAI报告：中国2025年贡献30个杰出模型，差距正在缩小
- 半导体设备国产化率从15%跃升至35%，刻蚀/薄膜沉积突破40%

**应用层：AI从能理解到能行动**

2026年关键转折：AI从「能理解」升级到「能行动」。智能体具备感知-决策-执行-优化完整闭环，成为可自主完成任务的数字员工。华为AgentArts已落地600+智能应用、19000+数字员工，覆盖80%新业务系统。

**投资线索推导**

AI军备竞赛爆发 → 算力需求持续 → 电力/能源需求 → 功率半导体（英飞凌涨价）、MLCC高端需求旺盛。

封闭性博弈 → 国产替代加速 → 半导体设备国产化率突破35% → 成熟制程+先进封装（Chiplet）成为战略方向。

**对Jerry的意义**

Jerry关注MLCC产业链和半导体国产化，这条判断帮助他理解为什么高端MLCC价格可以逆势上涨（AI算力拉动），以及为什么国产半导体设备企业能够进入盈利兑现期（国产化率突破35%）。

MLCC两极分化格局（AI强、消费电子弱）本质上就是AI军备竞赛在被动元件市场的映射。

**延伸思考**

AI竞争进入三层分工格局：基础模型（竞争最激烈但商业化路径最模糊）、智能体（已经开始实际部署，华为19000+数字员工）、垂直应用（医疗、制造、金融正在快速渗透）。

2026年的关键变量是：推理侧算力需求是否持续爆发（决定了MLCC高端需求能否维持），以及国产芯片能否真正进入量产阶段（决定了半导体设备国产化能否从35%继续突破）。
