# 四大云厂商6000亿美元算力军备竞赛

**背景**

2026年5月1日，Meta、亚马逊、谷歌、微软四大云厂商相继发布季度财报，AI需求持续超预期，2026年资本开支指引全面上调，算力军备竞赛进入新高潮。四大厂商合计年度资本开支预算突破6000亿美元，超出市场预期近一倍。

**核心洞察**

算力军备竞赛的本质不是"买GPU"，而是"锁定AI时代基础设施主导权"的战略博弈。

**6000亿美元的逻辑**

6000亿美元不是随机数字。它代表的是：每一个主要云厂商都在以年度资本开支的量级投资AI基础设施——不是一次性投入，而是年度可持续支出。这意味着AI基础设施的投入是长期结构性的，而非短周期泡沫。

对比参考：2020年全球军费开支约2万亿美元，四大云厂商一年的AI资本开支就相当于全球军费的30%。这个量级的基础设施投入，在人类工业史上几乎没有先例。

**对不同产业的影响层级**

第一层（直接受益）：英伟达GPU（DGX系列）、台积电CoWoS封装、HBM内存——这些是算力军备竞赛的最直接受益者，因为无论谁赢得AI基础设施竞争，都需要这些核心元件。

第二层（间接受益）：MLCC被动元件（AI服务器需要更多MLCC）、先进封装测试、服务器液冷——这些受益于AI服务器单机用量增加和功耗提升。

第三层（生态影响）：AI应用层（软件开发、内容生成、自动驾驶）——算力成本下降后，AI应用的经济性将进一步提升。

**对Jerry的意义**

Jerry的MLCC产业研究和财经分析框架中，四大云厂商资本开支是一个领先指标：这一数字上升 → AI服务器产量增加 → MLCC需求增加 → 高端MLCC涨价。

2026年四大云厂商全面上调资本开支，直接验证了MLCC晨报表中"AI需求支撑高端MLCC价格温和上行"的判断。对于Jerry的产业投资研究，这是一个重要的自上而下验证信号。

**延伸思考**

6000亿美元军备竞赛的风险在于：如果AI应用变现速度不及预期（企业AI投入产出比低迷），资本开支可能面临削减。但当前的数据（GPT-5.5、Claude Opus 4.7等前沿模型持续迭代、AI编程自主完成率跃升至80%）显示，AI应用的实际需求正在快速跟上资本投入。

另一个值得关注的趋势：四大云厂商不只买算力，还在自研芯片（Google TPU、亚马逊Trainium、微软Maia）。算力自研化将对英伟达构成长期挑战，但短期内（2026-2027年）英伟达的GPU生态优势仍然难以撼动。
