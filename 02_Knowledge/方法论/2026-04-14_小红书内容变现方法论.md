### 2.2 内容类型分类

| 类型 | 触发条件 | 风格要求 |
|-----|---------|---------|
| **开盘方向型** | 每周一开盘前 | 结构清晰，数据支撑，方向明确 |
| **周度展望型** | 周末或周初 | 宏观框架，情景分析，操作建议 |
| **即时分析型** | 重大突发事件 | 快速反应，逻辑推演，结论明确 |
| **综合复盘型** | 周末或月末 | 个人视角，感悟为主，数据为辅 |

### 2.3 鲁迅白描风格标准（已验证有效）

**核心原则：**
- 不用emoji列表堆砌标题
- 像日记，有观察感
- 语气是亲身经历，不是一手编译
- 结构：个人视角 → 数据观察 → 判断

**已验证有效的表达方式：**
```
开头：这个周/这几天...
数据：具体数字（涨幅%、价格、人数）
判断：我觉得/我认为
结尾：就这样/周一见
```

**反面案例（不要这样写）：**
```
"🔥【重磅】XX发布了YY！太厉害了！✅官方出品 ✅质量保证"
```
