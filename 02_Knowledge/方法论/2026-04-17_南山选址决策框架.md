# 南山选址决策框架：红花岭工业上楼 vs 其他选项

## 背景
扩维原子正在评估南山落地的可能性，两个核心信息浮出水面：
1. **红花岭工业上楼项目**：南山区工信局主导，入驻企业需签产业承诺函（产值/固投/研发/税收承诺）
2. **汇通金控**：南山区政府投资平台（136亿规模），可投资入驻南山的科技企业

## 决策框架

### Step 1：量化承诺函压力

产业承诺函的核心是**数字谈判**。步骤：
1. 拿到正式文本（向南山区工信局索取）
2. 逐项确认指标定义（产值=合并报表or单体？固投=含设备or含研发？）
3. 压力测试：按保守/中性/乐观三情景测算能否达标
4. 谈判空间评估：承诺数字是否可协商调整

### Step 2：政策收益量化

| 收益项 | 量化方法 |
|--------|---------|
| 租金优惠 | 对比市场价差×面积×年限 |
| 税收优惠 | 按实际利润×（减免税率）×年限折现 |
| 政策补贴 | 直接叠加计算 |
| 融资便利 | 汇通金控投资→股权稀释比例折算 |

### Step 3：汇通金控对接优先级排序

| 接触方式 | 优先级 | 说明 |
|---------|--------|------|
| 参加南山区路演会 | ⭐⭐⭐ | 直接面对投资团队，效果最好 |
| 联系汇通金控BP对接 | ⭐⭐ | 需要提前准备完整BP |
| 通过南山创业投资接触 | ⭐⭐ | 专注早期项目 |
| 直接联系工信局 | ⭐⭐⭐ | 先了解入驻条件和承诺函细节 |

### Step 4：选址对比矩阵

建议从5个维度打分（满分10分）：
1. **租金成本**（含政策优惠）
2. **产业链协同**（周边是否有上下游）
3. **融资便利度**（政府平台可触达性）
4. **人才吸引力**（地理位置、生活配套）
5. **政策稳定性**（历史兑现记录）

## 关键风险控制
- 承诺函数字**宁低勿高**：完成率比数字好看更重要
- **免责条款**：要求加入"不可抗力/市场重大变化"协商调整条款
- **退出机制**：如果5年后无法达标，场地和设备的退出成本是多少？
